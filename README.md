En este trabajo vas a encontrar mi curriculum realizado en Html5, 
donde fusione esta herramienta junto con CSS y Flexbox. 
La información presentada se enfoca principalmente en mostrar mis conocimientos y experiencias en el mundo del diseño,
asi como también mi recorrido profesional en diferentes sectores.
El trabajo consta de un navegador con mi mail para que sea más facil contactarme, y si tienen mayor curiosidad,
pueden ver mis otros proyectos en el mismo sector, 
también hay una imagen introductoria (logotipo realizado para mi portfolio personal) dos secciones de experiencias y conocimientos,
una sección con datos sobre mí, mi fotografía en el mismo y finalmente mi proyección a futuro en un apartado, dando un aspecto de cita
El tamaño de pantalla utilizado es de 1366x768 px y el editor de código utilizado es Visual Studio Code.
